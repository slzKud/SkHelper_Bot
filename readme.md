
# SkHelper_Bot

自己使用的TeleGram机器人,地址[点我](https://t.me/SkHelper9_Bot)

## 现在有的功能
    * ping/pong
    * 查询云服务器余额(支持Conoha和CloudCone)
    * ip查询
    * 一些常见的比如exec,eval
    * Conoha及CloudCone下线警告
    
## 待填坑
    * 基于HTTP的服务器下线警告
    * Vultr/TX云/阿里云服务器线上查询
    * 闹钟/早晨提醒
    * 学校通知提醒
    * 常用的备忘录
    * etc
## 安装
    *安装python3及依赖
    *配置config.example.py并改名为config.py
    *运行bot.py
    
## 需要的依赖
    * python-telegram-bot
    * requests
